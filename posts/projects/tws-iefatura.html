TWS ie-Fatura

Date: 2018
Topics: C# Accounting Webscraping
Website

I have developed a software to help accounts to manage the invoices for things a company buys. They have to be included in the accounting software.

For this pourpose, I have developed ie-Fatura. It logs in to the portuguese tax agency e-fatura website, and scrapes the invoices' data, which is saved locally. The user can then review them, and categorize them for accounting purposes.

It was developed in C#, using WinForms for the GUI, and the webscraping was done by reverse engineering the website's API. The data is saved in a JSON files.
