Converting VHS to digital format

Topics: vhs easycap linux
This is our first Post!

Many of us have old VHS tapes around with precious videos not backed up in digital form. This is a problem: you should always have backups of your things, VHS tapes' quality degrades of time, and you will rarely watch them because it's an hassle.

I will be sharing one easy way to convert them if you have a computer running Linux (you can use a live USB with Linux instead). I chose to use Linux because the drivers for the easycap adapter work without any effort there.

I am using

  • Computer running Linux
  • VHS player
  • Easycap adapter
  • (optional, in case your VHS player doesn't output composite video) SCART to composite adapter, configured as inputOUoutput

After connecting everything, which is pretty straight-forward, you are ready to start.

First, install FFmpeg,

On Ubuntu (or other apt-get based distro) sudo apt install ffmpeg

On Fedora (or other dnf based distro) sudo dnf install ffmpeg

You can open up VLC, chose capture device, and test weather you can see the output of the DVR, as well as hear the sound. It gets configured as a video device, in my case /dev/video0, and as an audio input, in my case hw:2,0.
